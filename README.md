# 简介		

荔知AI助手，是由 Brath 个人开发的一款全面 AIGC 站点，包含了市面上常用的各类 LLM 大模型，支持 StableDiffusion AI 绘画，目前稳定运营超过六个月，点击 https://brath.cn 立即加入荔知，畅享 AI 世界！
